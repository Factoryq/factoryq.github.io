Ahrefs - das Backlink Analyse Tool

Ahrefs ist eine Komplettlösung für Webmaster, die verschiedene Tools zur Analyse und Optimierung von Webseiten bietet. Verfügbar ist die Ahrefs SEO Suite als kostenloses Modell und als Premium-Version. Es gibt eine API und ein Affiliate-Programm. Die Premium-Modelle starten bei 79 US-Dollar pro Monat und gehen bis zur Enterprise-Variante mit 499 US-Dollar hoch. Ahrefs gewährt bei einem Jahres-Abonnement auch großzügige Rabatte. Dank der grafischen Darstellungen ist es auch für Einsteiger ein leicht zu bedienendes Tool. In der Ahrefs SEO Suite ist ein Site Explorer, ein Keyword Analyse Tool, ein Backlink Report, Domain Vergleich, Batch Analyse und ein Vergleichsanalyse Tool enthalten.

Site Explorer
Der Site Explorer von Ahrefs kann als Herzstück betrachtet werden. Er listet alle wichtigen Backlinkdaten auf. Hier kann gewählt werden, ob die Auswertung auf Hauptdomain, Subdomain oder URL-Ebene erfolgen soll. Sehr übersichtlich gehalten sind hier die Anzahl der Backlinks, IP-Pop und Domain-Pop, die Anzahl der .gov und .edu-Links und die Anzahl der Nofollow-Links aufgeführt. In einer Tabelle werden die entsprechenden Backlinks angezeigt. Hier ist auch ersichtlich, ob es sich um einen Textlink, einen Imagelink oder um eine Weiterleitung handelt.

 

SERPs Analytics
Mit den SERPs Analytics kann die Position einer Webseite ermittelt werden. Die sehr umfangreiche Datenasis wird alle 15 Tage neu ermittelt. So erhält der Webmaster immer die brandaktuellen Daten für die Länder Deutschland, USA, Großbritannien, Frankreich, Russland, Spanien, Italien, Australien und Brasilien.

 

Reports
Ahrefs erstellt für die eigene Domain kostenlos Reports. Ähnlich wie auch bei den Google Webmaster Tools gibt es auch hier verschiedene Möglichkeiten, die Webseite zu verifizieren. Ist die Webseite authentifiziert, hat der Webmaster Zugriff auf die ganze Datenpalette. Bei den Reports hat der Inhaber der Webseite alle wichtigen Daten auf einen Blick. Außerdem besteht die Möglichkeit, die Backlinks auf Seiten- und Subdomainebene zu betrachten.

 

API und Export von Ahrefs
Ahrefs bietet die Möglichkeit, die meisten Tabellen im CSV Format zu exportieren. Ein sehr großer Vorteil von Ahrefs ist, dass der Webmaster bei jedem Paket Zugang zur API hat. Für einen US Dollar erhält der Betreiber der Webbseite 1000 API Units. Eine API Unit wird für 10 Backlinks gerechnet. 10.000 Backlinks entsprechen also 1 US Dollar.

 

Fazit
Die Vorteile von Ahrefs liegen klar auf der Hand: die grafische Darstellung der SEO Suite ist sehr ausgereift und gibt einen detaillierten Überblick über alle relevanten Informationen.. Das Team bei Ahrefs ist sehr kundenfreundlich und bietet kompetenten Support. Die Ahrefs SEO Suite hat gute Karten, sehr bald ganz oben mitzuspielen.
