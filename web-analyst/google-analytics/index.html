Google Analytics

Mit Google Analytics, dem Analyse-Tool des Suchmaschinengiganten, ist es möglich, nicht nur Umsatz und Konversionen zu messen, sondern auch zu erfahren, wie der Benutzer auf die jeweilige Webseite gekommen ist und wie daraus ein regelmäßiger Besucher werden kann. 

Analyse Tools
Echtzeitberichte - Mit den Echtzeitberichten kann eingesehen werden, was den Besucher dazu veranlasst hat, die Seite anzuklicken. Hier kann auch eingesehen werden, welche neuen Inhalte sich wie auf die Besucher auswirken.

Benutzerdefinierte Berichte - Hier kann über ein Dashboard festgelegt werden, was den Webseiteninhaber genau interessiert. Diese Werte werden dann angezeigt. So muss sich der Anwender nicht durch Daten lesen, die er für seine Zwecke nicht benötigt.

Visualisierung - Google Analytics bietet eine Vielzahl von grafischen Tools. So kann der Besucherfluss und die In-Page-Analyse grafisch dargestellt werden 

Informationsaustausch - Mit diesem Werkzeug wird es möglich, dass das Keyword-Team mit dem SEO- und Content-Team reibungslos zusammenarbeiten kann.

API und Anpassungen - Google Analytics kann mit Hilfe von Anwendungen, Plugins und anpassbaren Funktionen ganz einfach in den unternehmensinternen Arbeitsablauf eingebunden werden.

 

Analytics für Content
Website-Suche - Mit dem Website-Suche Tool von Google Analytics kann festgestellt werden, wonach Besucher einer Webseite wirklich suchen. 

In-Page-Analyse - Mit der In-Page-Analyse ist es möglich, den Weg des Besuchers auf der Webseite nachzuvollziehen. 

Website-Geschwindigkeit - Lange Wartezeiten beim Aufbau einer Website wirken sich kontraproduktiv auf den Besucher aus. Google Analytics zeigt eventuelle Probleme auf, die so gezielt behoben werden können.

Ereignis-Tracking - Der Webseiteninhaber kann hier einsehen, wie oft zum Beispiel ein Flash- oder AJAX-Element genutzt wird oder auch eine Broschüre oder eine PDF-Datei heruntergeladen wird. 

AdSense effektiv nutzen - Mit Google Analytics können AdSense-Daten problemlos importiert werden.

 

Analytics für mobilen Traffic
Analytics für mobile Apps – hier kann der ganze Weg von der Entdeckung der App über den Download is hin zum konkreten Interesse verfolgt werden.

Optimale Nutzererfahrung – mit Hilfe von Karten kann hier genauestens bestimmt werden, wo sich der Zugriff von mobilen Endgeräten erhöht und wo er sinkt. Es werden genaue Daten über das jeweilige Gerät und das System geliefert.

Leistung mobiler Anzeigen messen – mit Hilfe von Google Mobilen Anzeigen können potenzielle Interessenten auf die jeweilige Webseite aufmerksam gemacht werden. Hier können die Konversionen mobiler Nutzer genau verfolgt werden.

 

Berichte zu sozialen Netzwerken
Auswirkungen sozialer Medien messen – der Conversions-Bericht zeigt die Anzahl und den finanziellen Wert von Konversionen. 

Soziale Quellen ermitteln – hier wird aufgezeigt, welchen Weg der Besucher gegangen ist, um aus Communities und sozialen Netzwerken auf die jeweilige Webseite zu gelangen.

Wo teilt der Besucher welche Inhalte? - der Bericht „Plugins für soziale Netzwerke“ zeigt dem Website-Betreiber genau an, wo welcher Inhalt der Webseite wie oft geteilt wurde.

 

Analytics für Werbung
Suchmaschinenmarketing – mit Google Analytics kann das Besucherverhalten mit der SEM-Kampagne verknüpft, die Gebote verbessert und die Keyword-Auswahl optimiert werden.

Den Einfluss von Display-Kampagnen ermitteln – der Website-Betreiber kann so herausfinden, durch welche Display-Kampagnen Besucher auf die jeweilige Website aufmerksam wurden.

Werbekampagnen und Multichannel-Trichter – mit dem Zusammenspiel dieser beiden Faktoren kann erkannt werden, welche Rolle die einzelnen Kanäle spielen.

Kampagnenauswertung – so kann der Betreiber der Website genau feststellen, welche Kampagnen sich lohnen und welche nicht.
